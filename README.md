# 🐟 ModelF - Deep Learning Fish Classifier

A TensorFlow/Keras-based machine learning project that classifies fish species. The model is trained on a high-resolution dataset containing over 24,000 images across 82 fish classes.

## 🚀 Features
- **Colab Notebook**: Ready-to-run Jupyter notebook (`fish.ipynb`) with Drive mounting.
- **Model Architecture**: Convolutional Neural Network (CNN) built using Keras `models` and `layers`.
- **Pre-processing**: Uses `image_dataset_from_directory` with batch resizing (640x640) for high-accuracy training.

## 🛠️ Tech Stack
- Python 3
- TensorFlow & Keras
- Jupyter Notebook
